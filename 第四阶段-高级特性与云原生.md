# 第四阶段-高级特性与云原生

## 一、必背知识点

### 1. RabbitMQ消息队列

**核心概念：**
- 消息队列：用于在应用间异步传递消息的中间件
- Exchange：消息交换机，负责接收消息并路由到队列
- Queue：消息队列，存储消息直到被消费
- Binding：绑定关系，将Exchange和Queue连接起来
- Routing Key：路由键，决定消息路由到哪个队列

**消息模式：**
- Direct模式：根据Routing Key精确匹配路由
- Fanout模式：广播模式，将消息发送到所有绑定的队列
- Topic模式：主题模式，支持通配符匹配
- Headers模式：根据消息头属性路由

**消息可靠性：**
- 消息持久化：将消息保存到磁盘
- 消息确认：消费者确认消息处理完成
- 死信队列：处理无法消费的消息
- 事务机制：保证消息发送的原子性

### 2. 异步处理与任务调度

**@Async异步注解：**
- 标记方法为异步执行
- 需要配置@EnableAsync启用异步支持
- 支持自定义线程池
- 可设置异步方法返回值

**@Scheduled定时任务：**
- fixedRate：固定频率执行，不考虑上一次执行时间
- fixedDelay：固定延迟执行，从上一次执行完成后开始计时
- cron：使用Cron表达式定义执行时间
- initialDelay：首次执行的延迟时间

**Cron表达式格式：**
- 秒 分 时 日 月 周
- *：任意值
- ?：不指定值（日和周互斥）
- -：范围（如1-5）
- ,：列表（如1,3,5）
- /：间隔（如*/5表示每5个单位）
- L：最后（如周L表示最后一周）

### 3. 邮件发送功能

**JavaMail核心组件：**
- JavaMailSender：邮件发送接口
- SimpleMailMessage：简单邮件消息
- MimeMessage：MIME格式邮件消息
- MimeMessageHelper：MIME消息辅助类

**邮件类型：**
- 简单文本邮件：纯文本内容
- HTML邮件：支持HTML格式
- 附件邮件：包含文件附件
- 模板邮件：使用模板引擎生成内容

**邮件配置：**
- SMTP服务器地址和端口
- 发件人账号和密码
- 是否使用SSL/TLS加密
- 连接超时和读取超时

### 4. 应用监控

**Spring Boot Actuator：**
- 提供生产级监控端点
- 支持HTTP和JMX访问
- 可自定义端点
- 支持端点安全控制

**常用端点：**
- /actuator/health：健康检查
- /actuator/info：应用信息
- /actuator/metrics：指标监控
- /actuator/env：环境信息
- /actuator/loggers：日志配置
- /actuator/threaddump：线程快照

**健康检查：**
- 数据库连接检查
- 磁盘空间检查
- 外部服务检查
- 自定义健康指标

### 5. Docker容器化

**Docker核心概念：**
- 镜像（Image）：应用的只读模板
- 容器（Container）：镜像的运行实例
- 仓库（Repository）：存储和分发镜像
- Dockerfile：构建镜像的脚本文件

**Dockerfile指令：**
- FROM：指定基础镜像
- COPY/ADD：复制文件到镜像
- RUN：执行命令
- CMD/ENTRYPOINT：容器启动命令
- ENV：设置环境变量
- EXPOSE：暴露端口
- VOLUME：定义数据卷

**镜像优化：**
- 使用多阶段构建减小镜像大小
- 选择合适的基础镜像（如alpine）
- 合并RUN命令减少层数
- 利用Docker缓存机制

### 6. Docker Compose编排

**Compose核心概念：**
- 服务（Service）：定义容器配置
- 网络（Network）：容器间通信
- 卷（Volume）：数据持久化
- 环境变量：配置注入

**常用指令：**
- docker-compose up：启动服务
- docker-compose down：停止并删除服务
- docker-compose ps：查看服务状态
- docker-compose logs：查看日志
- docker-compose exec：进入容器

**编排优势：**
- 一键启动多容器应用
- 统一管理服务配置
- 自动处理服务依赖
- 支持环境变量配置

### 7. 云原生应用

**云原生特征：**
- 容器化：使用容器打包应用
- 微服务：应用拆分为小型服务
- 动态编排：自动化部署和扩展
- 持续交付：自动化CI/CD流程

**GraalVM原生镜像：**
- AOT编译：提前编译为机器码
- 快速启动：毫秒级启动时间
- 低内存：显著降低内存占用
- 接近原生性能

**Kubernetes核心概念：**
- Pod：最小部署单元
- Service：服务发现和负载均衡
- Deployment：应用部署管理
- ConfigMap/Secret：配置管理

### 实操1：RabbitMQ消息队列集成

**步骤1：添加依赖**

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-amqp</artifactId>
</dependency>
```

**步骤2：配置RabbitMQ连接**

```properties
spring.rabbitmq.host=localhost
spring.rabbitmq.port=5672
spring.rabbitmq.username=guest
spring.rabbitmq.password=guest
spring.rabbitmq.publisher-confirm-type=correlated
spring.rabbitmq.publisher-returns=true
```

**步骤3：创建配置类**

```java
@Configuration
public class RabbitMQConfig {
    
    @Bean
    public Queue helloQueue() {
        return new Queue("hello.queue");
    }
    
    @Bean
    public DirectExchange helloExchange() {
        return new DirectExchange("hello.exchange");
    }
    
    @Bean
    public Binding helloBinding() {
        return BindingBuilder
            .bind(helloQueue())
            .to(helloExchange())
            .with("hello.routing.key");
    }
}
```

**步骤4：创建消息生产者**

```java
@Service
public class MessageProducer {
    
    @Autowired
    private RabbitTemplate rabbitTemplate;
    
    public void sendMessage(String message) {
        rabbitTemplate.convertAndSend(
            "hello.exchange",
            "hello.routing.key",
            message
        );
    }
}
```

**步骤5：创建消息消费者**

```java
@Component
public class MessageConsumer {
    
    @RabbitListener(queues = "hello.queue")
    public void receiveMessage(String message) {
        System.out.println("Received message: " + message);
    }
}
```

**步骤6：测试消息发送**

```java
@RestController
@RequestMapping("/api/message")
public class MessageController {
    
    @Autowired
    private MessageProducer messageProducer;
    
    @PostMapping("/send")
    public String sendMessage(@RequestParam String msg) {
        messageProducer.sendMessage(msg);
        return "Message sent: " + msg;
    }
}
```

### 实操2：异步处理与定时任务

**步骤1：启用异步支持**

```java
@SpringBootApplication
@EnableAsync
@EnableScheduling
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

**步骤2：配置异步线程池**

```java
@Configuration
public class AsyncConfig {
    
    @Bean(name = "taskExecutor")
    public Executor taskExecutor() {
        ThreadPoolTaskExecutor executor = new ThreadPoolTaskExecutor();
        executor.setCorePoolSize(5);
        executor.setMaxPoolSize(10);
        executor.setQueueCapacity(25);
        executor.setThreadNamePrefix("async-");
        executor.initialize();
        return executor;
    }
}
```

**步骤3：创建异步服务**

```java
@Service
public class AsyncService {
    
    @Async("taskExecutor")
    public CompletableFuture<String> asyncMethod(String name) {
        try {
            Thread.sleep(2000);
            return CompletableFuture.completedFuture("Hello, " + name);
        } catch (InterruptedException e) {
            return CompletableFuture.failedFuture(e);
        }
    }
}
```

**步骤4：创建定时任务**

```java
@Component
public class ScheduledTasks {
    
    @Scheduled(fixedRate = 5000)
    public void fixedRateTask() {
        System.out.println("Fixed rate task: " + new Date());
    }
    
    @Scheduled(fixedDelay = 5000)
    public void fixedDelayTask() {
        System.out.println("Fixed delay task: " + new Date());
    }
    
    @Scheduled(cron = "0 0 * * * ?")
    public void cronTask() {
        System.out.println("Cron task (every hour): " + new Date());
    }
}
```

**步骤5：测试异步调用**

```java

@RequestMapping("/api/async")
public class AsyncController {
    
    @Autowired
    private AsyncService asyncService;

    }
}

### 实操3：邮件发送功能

**步骤1：添加依赖**

**步骤2：配置邮件服务器**

```

**步骤3：创建邮件服务**

```java
@Service
public class EmailService {
    
    @Autowired
    private JavaMailSender mailSender;
    
    public void sendSimpleEmail(String to, String subject, String text) {
        SimpleMailMessage message = new SimpleMailMessage();
        message.setTo(to);
        message.setSubject(subject);
        message.setText(text);
        mailSender.send(message);
    }
    
    public void sendHtmlEmail(String to, String subject, String htmlContent) 
        throws MessagingException {
        MimeMessage message = mailSender.createMimeMessage();
        MimeMessageHelper helper = new MimeMessageHelper(message, true);
        helper.setTo(to);
        helper.setSubject(subject);
        helper.setText(htmlContent, true);
        mailSender.send(message);
    }
    
    public void sendEmailWithAttachment(String to, String subject, 
            String text, String attachmentPath) throws MessagingException {
        MimeMessage message = mailSender.createMimeMessage();
        MimeMessageHelper helper = new MimeMessageHelper(message, true);
        helper.setTo(to);
        helper.setSubject(subject);
        helper.setText(text);
        helper.addAttachment("attachment.pdf", new File(attachmentPath));
        mailSender.send(message);
    }
}
```

**步骤4：创建邮件控制器**

```java
@RestController
@RequestMapping("/api/email")
public class EmailController {
    
    @Autowired
    private EmailService emailService;
    
    @PostMapping("/send")
    public String sendEmail(@RequestParam String to, 
                           @RequestParam String subject,
                           @RequestParam String text) {
        emailService.sendSimpleEmail(to, subject, text);
        return "Email sent successfully";
    }
}
```
