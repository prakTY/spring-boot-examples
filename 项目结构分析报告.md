## 📦 项目分类总结

### 1. 基础入门项目（5个）

#### spring-boot-hello
- **版本：** Spring Boot 3.0
- **功能：** 最基础的 Hello World 示例
- **技术点：**
  - Spring Boot 项目结构
  - @SpringBootApplication 注解
  - @RestController 注解
  - 基本的 REST API
- **适用场景：** Spring Boot 入门学习

#### spring-boot-helloWorld
- **版本：** Spring Boot 3.0
- **功能：** 单元测试示例
- **技术点：**
  - @SpringBootTest 注解
  - MockMvc 测试
  - 单元测试编写
- **适用场景：** 学习 Spring Boot 测试

#### spring-boot-banner
- **版本：** Spring Boot 3.0
- **功能：** 自定义启动 Banner
- **技术点：**
  - 自定义 Banner 文件
  - Banner 配置选项
  - 启动界面美化
- **适用场景：** 个性化应用启动

#### spring-boot-commandLineRunner
- **版本：** Spring Boot 3.0
- **功能：** 启动时初始化资源
- **技术点：**
  - ApplicationRunner 接口
  - CommandLineRunner 接口
  - 执行顺序控制
- **适用场景：** 应用启动初始化

#### spring-boot-package
- **版本：** Spring Boot 3.0
- **功能：** 打包和部署示例
- **技术点：**
  - Jar 包打包
  - War 包打包
  - 多环境配置
  - 启动脚本编写
- **适用场景：** 生产环境部署
